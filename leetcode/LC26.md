# 26. Remove Duplicates from Sorted Array

[풀이 링크](LC26.java )  

## 문제 설명
오름차순으로 정렬된 정수 배열 `nums`가 주어진다.  
중복해서 등장하는 원소를 하나만 남겨, 모든 원소가 한 번만 등장하도록 만들고, 남은 배열의 길이를 구하라.  

**제한 사항**  
`1 <= nums.length <= 30,000`  
`각 원소의 값은 -100 이상, 100 이하이다.`
