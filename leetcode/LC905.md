# 905. Sort Array By Parity
[문제 링크](https://leetcode.com/problems/sort-array-by-parity/ )  
[풀이 링크](LC905.java )  

## 문제 설명
정수 배열 `nums`가 주어진다.  
배열의 모든 짝수를 배열의 앞쪽으로 옮겨서, 모든 짝수가 모든 홀수보다 앞에 있도록 만들어라.  
이 조건을 만족하는 아무 배열이나 리턴하라.  

**예시**
```
[3,1,2,4] --> [2,4,3,1],[4,2,3,1],[2,4,1,3],[4,2,1,3]
[0] --> [0]
```

**제한 사항**  
* `1 <= nums.length <= 5,000`  
* `0 <= nums[i] <= 5,000`  

## 풀이
새로운 배열의 왼쪽과 오른쪽을 가리키는 각각의 포인터를 만들어, 배열을 순회하면서 홀수/짝수 여부를 판단해 짝수는 왼쪽에, 홀수는 오른쪽에 담으며 새로운 배열에 옮긴다.
