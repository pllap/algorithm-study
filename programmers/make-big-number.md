# 큰 수 만들기
[문제 링크](https://programmers.co.kr/learn/courses/30/lessons/42883 )  
[풀이 링크](make-big-number.py )  

## 문제 설명
어떤 숫자에서 k개의 수를 제거했을 때 얻을 수 있는 가장 큰 숫자를 구하려 한다.  
문자열 형식으로 `number`가 주어지고, 제거할 수의 개수 `k`가 주어진다.  
`k`개를 제거했을 때 만들 수 있는 가장 큰 숫자를 문자열 형태로 반환하는 함수를 작성하라.  

**예시**
```
1924, 2 --> 94
54321, 2 --> 543
12345, 2 --> 345
991991991, 4 --> 99999
```

## 풀이
인덱스를 하나씩 증가시켜 가면서, 현재 인덱스의 수보다 뒤 인덱스의 수가 더 크면 해당 인덱스를 제거한다. 제거했으면 인덱스를 0으로 돌려 처음부터 다시 검사한다.  
위 과정을 k번 반복한다.  

최적화 때문에 사흘 동안 붙잡고 있던 문제다.
